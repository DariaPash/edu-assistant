# EDU‑ASSISTANT

Гибкий LLM‑ассистент для обучения, который умеет отвечать на вопросы по математике, истории и другим предметам.  
Вы сами выбираете модель (облачное API или локальный Ollama), назначаете роль (учитель математики, историк и т.д.) и задаёте формат ответа (краткий или развёрнутый с упражнениями).  

---

## Возможности

- **Несколько LLM‑провайдеров** — подключайтесь к любому OpenAI‑совместимому API или к локальному Ollama.
- **Настраиваемые роли** — каждая роль имеет свою системную инструкцию (например, *«ты учитель математики, не используй LaTeX»* или *«ты учитель истории, отвечай кратко с датами»*).
- **Гибкие шаблоны ответов** — один шаблон даёт развёрнутый ответ с пошаговым объяснением и упражнениями, другой — максимально сжатую справку в одно предложение.
- **Единый конфиг** — все настройки моделей, ролей и промптов хранятся в `config.yml`.
- **Простой запуск** — достаточно выполнить `main.py`, чтобы проверить работу ассистента с вашими параметрами.
